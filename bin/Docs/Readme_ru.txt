ОПИСАНИЕ
--------
Программа для освобождения страниц памяти системы. Освобождаются 
системный кэш, модифицированные и простаивающие страницы памяти. В 
результате использования программы, использование памяти уменьшается на 
~25%. Для полноценной работы программы необходимы права администратора.

ПРЕДУПРЕЖДЕНИЕ
--------------
Программа использует недокументированные возможности системы (Native
API) для очистки памяти, которые появились начиная с Windows Vista, 
поэтому некоторые методы очистки не доступны в более ранних версиях 
Windows.

СИСТЕМНЫЕ ТРЕБОВАНИЯ
--------------------
- OS: Windows XP/Server/Vista/Seven/8
- CPU: 500 mhz
- HDD: 10 mb
- RAM: 32 mb

ВОЗМОЖНОСТИ
-----------
- Отображение информации об используемой памяти
- Отображение информации об используемой памяти в трее (значок, всплывающая подсказка)
- Всплывающие подсказки по событиям
- Сравнение памяти до и после очистки
- Настраиваемая автоочистка/очистка памяти
- Поддержка локализации
- Простой и понятный интерфейс - ничего лишнего
- Доступен полный исходный код программы
- Портативность, все настройки хранятся в папке с программой
